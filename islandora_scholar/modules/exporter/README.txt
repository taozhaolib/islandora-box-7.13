SUMMARY
-------

Citation Exporter

Exports a collection of citations to either an RIS/RTF/PDF file.

REQUIREMENTS
------------

The following Drupal modules are required:
 * islandora
 * citeproc
 * bibutils

INSTALLATION
------------

Enable the module in the admin/modules page.
